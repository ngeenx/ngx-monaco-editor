Editor Theme Switch Exmaple

  • Warning #1: Monaco editor supports only one global theme. When you changed any instance's theme, all monaco instances will be destroyed and recreated with the new theme. This is a limitation of monaco editor ( See issue #338 ).
  • Warning #2: Yes, there are some hacks but monaco editor still getting updates and you should be carteful while using these short term fixes.
  • Warning #3: Some textmate grammars and themes may not properly work with dark or light editor themes. You should adjust them accordingly.
